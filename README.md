# terraform-aws-main-auy1105-cr


Repositorio Terraform para desplegar recursos en AWS mediante módulos remotos.

## Propósito

Este repositorio proporciona la definición de infraestructura como código para crear una VPC con subredes públicas y privadas, instancias EC2 con servidor web, balanceador de carga y un bucket S3.
